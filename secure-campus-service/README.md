# Secure Containerized Campus Service

## Project Overview
This project demonstrates a secure, multi-container architecture using Docker and Ubuntu Server.

## Architecture Highlights
- **Web Server:** Nginx container serving the campus dashboard.
- **Database:** MySQL 8.0 container for persistent data storage.
- **Storage:** Docker Volumes are used to ensure data persistence.
- **Networking:** Isolated bridge network for secure communication between services.
